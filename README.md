# Stock Portfolio Tracker
A simple Python application that calculates the total investment value of a stock portfolio using user-input stock names and quantities with predefined stock prices. The project demonstrates dictionaries, loops, arithmetic operations, user input and file handling by saving portfolio details to a text file.

## Problem Statement
Create a simple stock portfolio tracker that calculates the total investment value based on user-entered stock names and quantities using predefined stock prices.

## Key Features
- User enters stock names and quantities.
- Uses a predefined dictionary to store stock prices.
- Calculates individual and total investment value.
- Displays portfolio summary.
- Saves portfolio details to a .txt file.

## Concepts Used
- Dictionary
- Input/Output
- Basic Arithmetic
- File Handling
